# 宝可梦调试工具

## 需求

现有的在托盘上的调试功能不够用，需要更多的调试入口。
这些入口只在dev模式下可见，在build production版本的时候，关掉调试入口

## 调试面板

1. 开启调试面板后，在面板中可以任意调整玩家的所有数据，不同类型的数值请选用适合的组件来方便我的使用，以下是面板需要提供的能力
   分为两个tab，宝可梦，背包

   宝可梦tab
   1. 添加和删除持有宝可梦
   2. 点击持有的宝可梦，进入宝可梦编辑界面，可以改变宝可梦的等级和持有道具。
   
   背包tab
   1. 添加和删除背包中的道具
   
2. 面板footer有确认和取消两个按钮，在数据改动后，点击确认再生效。
